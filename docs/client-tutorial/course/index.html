Course

A course contains two types of roles:

  • Instructor (creator)
  • Student (learner)

Create Course​

  1. Click the Course button and select Create Course on the menu bar.
  2. Enter the course name and description.
  3. Then click Submit to create the course.

Join Course​

The only way to join a course is by invitation.

Check the invitation list to see if you have been invited to a course.

  1. Click the Course button and select Course Invitation on the menu bar.
  2. Then the invitation list will be shown.
  3. You can choose Accept or Reject the invitation.

Instructor​

Publish Announcement​

Instructor can publish announcement to all students in the course.

  • Click the Publish New Announcement button.

The announcement contains:

  • Name
  • Publish Time
  • Content

Every one can view the announcement content by clicking the View button.

Publish Assignment​

Instructor can publish assignment to all students in the course.

  • Click the Publish New Assignment button.

The assignment contains:

  • Name
  • Attachment
  • Deadline

The instructor can view the submitted assignment by clicking the View Submission button.

Publish Resource​

Instructor can publish resource to all students in the course.

  • Click the Publish New Resource button.

The resource contains:

  • Name
  • Type
  • Publish Date
  • File Size

Everyone can download the resource by clicking the Download button.

Invite new student​

Instructor can invite new student to the course.

  • Click the Invite New Member button.

  • Then a pop-up window will be shown.

  • Enter all students' id you want to invite line by line.

  • Then click Submit to send the invitation.

Delete Course​

Instructor can delete the course.

  • Click the Delete Course button.
  • Then click Confirm to delete the course.

Student​

Student's main page contains:

  • Course Announcement
  • Course Assignment
  • Course Resource

Announcement​

  • Student can view the announcement by clicking the View button.

Assignment​

  • Student can download the attachment by clicking the attachment link.
  • Student can submit the assignment by clicking the Submit Assignment button.
  • There is a Status column to show the submission status.

Resource​

  • Student can download the resource by clicking the Download button.

Member List​

Click the member list button to view the member list.
