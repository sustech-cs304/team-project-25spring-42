## Installation
1. Download the latest release from our [Releases page](https://github.com/sustech-cs304/team-project-25spring-42/releases)
2. Run the installer (Windows: .exe, macOS: .dmg)
3. Launch AIDE and authenticate with:
   - Google account
   - GitHub account
